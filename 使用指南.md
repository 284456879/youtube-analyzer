# YouTube欧美热门视频分析工具 - 使用指南

> 专为搬运到小红书/抖音设计，快速找到适合本地化的欧美热门内容

## 🎯 核心功能

- ✅ 筛选欧美地区（美国）热门视频
- ✅ 智能过滤适合短视频平台的时长（1-15分钟）
- ✅ 优先选择高互动率内容（更容易引起共鸣）
- ✅ 只抓取最近14天新鲜内容
- ✅ 自动导出Excel分析报表

## 📊 默认筛选条件（已优化）

| 条件 | 设置 | 说明 |
|------|------|------|
| **播放量** | ≥5万 | 有热度但不会太热门，适合搬运 |
| **互动率** | ≥2% | 观众喜欢，容易引起共鸣 |
| **视频时长** | 1-15分钟 | 适合剪辑成短视频 |
| **发布时间** | 14天内 | 保证内容新鲜 |
| **地区** | 美国 | 欧美地区热门内容 |
| **语言** | 英文 | 便于字幕翻译 |

## 🚀 快速开始

### 方式1：命令行运行

```bash
python youtube_analyzer.py
```

按提示选择：
1. **按关键词搜索** - 输入英文关键词
2. **分析指定频道** - 输入频道URL

### 方式2：代码调用

```python
from youtube_analyzer import YouTubeAnalyzer

# 创建分析器
analyzer = YouTubeAnalyzer('你的API密钥')

# 搜索分析
results = analyzer.analyze(
    input_type='keyword',
    input_value='cooking tips',  # 搜索关键词
    max_results=50,              # 最多分析50个视频
    min_views=50000,             # 最低5万播放
    min_engagement=2.0           # 最低2%互动率
)
```

## 🎬 搬运建议

### ✅ 适合搬运的视频特征
1. **教程类内容** - 步骤清晰，易于模仿
2. **视觉效果强** - 不依赖语言，画面就能看懂
3. **实用性高** - 解决生活问题
4. **互动率高** - 证明观众喜欢
5. **时长适中** - 1-15分钟，可剪辑成多个短视频

### ⚠️ 需要注意
1. **版权问题** - 建议二次创作，不要直接搬运
2. **本地化改造** - 加中文配音/字幕，适配国内审美
3. **时效性** - 14天内的内容更有新鲜感
4. **竞争分析** - 5-50万播放的视频竞争度适中

## 🔧 自定义配置

编辑 `config.json` 调整默认参数：

```json
{
  "analysis_settings": {
    "min_views": 50000,           // 最低播放量
    "min_engagement_rate": 2.0,   // 最低互动率

    "min_duration_seconds": 60,   // 最短时长(秒)

  }

```

## 📞 常见问题

**Q: 为什么找不到符合条件的视频？**
A: 可以降低要求，如 `min_views=30000`, `min_engagement=1.5`

**Q: 如何找到更多视频？**
A: 增加 `max_results` 参数到100，或尝试不同关键词

**Q: Excel在哪里？**
A: 默认保存在 `output/` 文件夹

**Q: 热度指数怎么算的？**
A: 公式：(播放×0.3 + 点赞×30 + 评论×15) / 发布天数
   更注重互动率，因为互动高的视频更有价值
